# React Login & Signup Template

A modern, animated login and signup flow built with React, Vite, and Framer Motion.

## Features

- **Modern UI**: Clean and responsive design with custom input components.
- **Page Transitions**: Smooth slide and fade animations between Login and Signup pages using `framer-motion`.
- **Routing**: Client-side routing with `react-router-dom`.
- **Asset Management**: SVG assets for icons and illustrations.

## Tech Stack

- React
- Vite
- Framer Motion
- React Router DOM

## Getting Started

1.  **Install dependencies:**

    ```bash
    npm install
    ```

2.  **Run the development server:**

    ```bash
    npm run dev
    ```

3.  **Open your browser:**
    Navigate to `http://localhost:5173` (or the port shown in your terminal).

## Project Structure

- `src/pages`: Contains `Login.jsx` and `Signup.jsx` pages.
- `src/components`: Reusable UI components like `InputField`.
- `src/assets`: Images and icons.
- `src/App.jsx`: Main application component with routing configuration.
